# Angular Learning Project

## About This Project
This project was created to learn Angular by building real applications through an online course. It covers the basics of Angular and how to build modern web applications.

---

## Purpose
- Learn Angular from the basics.
- Understand how Angular applications work.
- Practice building Single Page Applications (SPA).
- Improve TypeScript programming skills.

---

## Features
- Component-based Architecture
- Standalone Components
- Signals
- Data Binding
- Event Binding
- Routing
- Forms
- HTTP Requests
- Responsive Web Design

---

## Technologies Used
- Angular
- TypeScript
- HTML5
- CSS3
- Angular CLI
- Visual Studio Code
- Git & GitHub

---

## What I Learned
- Creating and managing Angular components.
- Communication between components.
- Using Signals for state management.
- Routing and navigation.
- Sending HTTP requests.
- Building and managing forms.
- Writing TypeScript with Angular.
- Developing scalable web applications.
